# 垃圾标书生成器
分四阶段自动生成垃圾标书，研究牲专用。标书生成分五个阶段：

- 提供一级标题，使用ai为一级标题进行拓展，生成二级标题。（输出文件 secondary_outline.md）
- 读取二级标题，为每个二级标题生成当前二级标题应该描写的重点。（keypoints/all_keypoints.md）
- 将生成的二级标题清洗，格式化。（cleaned_output.md）
